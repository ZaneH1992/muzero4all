# muzero4all


This is our MuZero implementation for [Stanford CS234](https://cs234.stanford.edu).

Setup:

    $ pip3 -m venv path/to/virtual
    $ source path/to/virtual/bin/activate
    $ pip3 install -r requirements.txt
